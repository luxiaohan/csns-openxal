Introduction - Features - Startup - Acknowledgements


Introduction:

This application manage the Magnet database of CSNS. Uploading and modifying actions are provided. When uploading magnet to the database success, backup will be done at the same time.


Features and Tips:

When using the application, magnet database connection is needed. The database is defined by other database application. A database connection configure file is provided at the resources directory. Backup directory is also needed. If there isn't a backup, uploading will be discarded. The application can also load magnet backup files in xml format which is defined by the Magnet class.


Start the Application:

The basic steps for starting the application:

  1. Launch the application.
  2. Connect the CSNS magnet database by select the connection item in the sub item of "Settings" menu.
  3. Fetch magnets from database or from local files. You can slect the magnet type before fetching. If you fetching from local files, multi-selection is allowed.
  4. You can also create a new empty magnet file by click the "New" item in the context menu of the table.
  5. You can edit a magnet data by click the "Edit" item in the context menu of the table. When the magnet is in editing model, magnetic excitation curve can be added or removed through context menu.
  6. Before uploading your magnet, default backup directory must be set by selecting the sub item of "Setting" menu.
  7. Select "Uploading" context menu item to upload the editing result or new magnet.
